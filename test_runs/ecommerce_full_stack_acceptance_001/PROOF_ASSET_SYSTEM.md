# PROOF_ASSET_SYSTEM

## Objective
Standardize proof capture so marketing claims stay credible, reusable, and compliant.

## Templates
1. Case study capture
2. Experiment write-up
3. SEO change log
4. Launch narrative
5. Sales proof summary
6. Testimonial evidence handling rules

## Evidence Handling Rules
- **Verified:** Only use attributable customer quotes with date/source.
- **Probable:** Aggregated insights can be used if data extraction method is documented.
- **Provisional:** Early anecdotal claims must be labeled as anecdotal and not generalized.

## Compliance Boundaries
- No disease-treatment claims.
- No guaranteed performance outcomes.
- Flag legal-sensitive phrasing for human approval.
