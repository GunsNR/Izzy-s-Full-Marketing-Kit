# BRAND_SYSTEM

## Objective
Create a differentiated, ownable brand system for hydration sticks with strong trust and performance cues.

## Name Options
1. PaceSalt
2. Hydrant Loop
3. Field Current
4. FluxHydrate

## Selected Name
**Field Current**
- **Probable:** Name supports active + professional identity simultaneously.

## Tagline Options
- "Hydration for the hours that matter."
- "Stay sharp. Stay in flow."
- "From desk to distance."

## Brand Promise
Reliable hydration ritual that supports consistent performance across work and training days.

## Voice / Tone Rules
- Crisp, practical, no pseudo-science.
- Coach-like clarity, not bro-science hype.
- Show uncertainty honestly when evidence is still developing.

## Message Pillars
1. Portable protocol, not random supplementation.
2. Transparent ingredient rationale.
3. Built for real schedules, not elite-only routines.

## Visual Direction
- Palette: Deep Navy (#0F1B2D), Electric Teal (#00BFA6), Cloud (#F4F7FA), Slate (#5D6B7A).
- Typography: Primary sans (Inter-like), accent condensed for performance callouts.
- Image style: Real-world movement + work contexts; avoid generic gym stock.
- Packaging brief: single-serve sachets with clear electrolyte grams and use-case icons.

## Dependencies
- Final packaging dielines from supplier.
- Compliance-approved claim language.

## Rollback
If name trademark risk appears, fallback to "PaceSalt" using same narrative pillars.

## Assumptions
- **Probable:** "Field Current" resonates with performance + professionalism.
- **Provisional:** Brand-name resonance and recall need audience testing.

## Source Clarity
- **Verified:** Voice, palette, and messaging align with strategy foundation from this run.
- **Provisional:** Trademark availability has not been validated in this environment.

## Next Action
Move brand system into website wireframes and conversion copy system.
