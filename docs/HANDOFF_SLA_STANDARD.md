# Handoff SLA Standard

## SLA Levels
- **Critical:** production-impacting issue, response < 4h, update every 8h.
- **High:** revenue-impacting but controlled, response < 1 business day.
- **Normal:** planned backlog work, response < 2 business days.

## Mandatory Handoff Fields
- owner
- objective
- KPI target
- due date
- dependencies
- risk level
- approval mode
- rollback trigger
