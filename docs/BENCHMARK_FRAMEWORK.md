# Benchmark Framework

## Objective
Detect improvements, regressions, and drift in skill performance and routing quality.

## Benchmark Families
1. Routing benchmarks (correct skill selection)
2. Workflow benchmarks (end-to-end output completeness)
3. Safety benchmarks (gate/risk compliance)
4. Evidence benchmarks (label correctness)

## Regression Rule
If benchmark score declines materially versus previous approved baseline, trigger safe mode and human review before rollout.
